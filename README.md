# RetailStore-Project
Retail Store Database using MySQL with customer and sales analysis
Retail Store Database Project

##  Description
This project is a Retail Store Database built using MySQL. It demonstrates core database concepts like table creation, primary key, foreign key relationships, data insertion, and different types of SQL joins.

##  Technologies Used
- MySQL
- SQL

##  Database Features
- Created multiple related tables
- Used *Primary Key* for unique identification
- Used *Foreign Key* to maintain relationships between tables
- Inserted sample data into tables
- Performed data retrieval using SQL queries

##  SQL Concepts Covered
- CREATE DATABASE & CREATE TABLE
- INSERT INTO
- SELECT queries
- Primary Key & Foreign Key
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL JOIN

##  Sample Operations
- Fetch customer and product details using JOINS
- Display combined data from multiple tables
- Analyze relationships between tables

##  How to Run
1. Open MySQL Workbench
2. Import or open the SQL. file
3. Execute all queries step by step
4. Run SELECT queries to view
